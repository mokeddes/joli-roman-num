# roman-numeral-converter

Converts integer numbers to roman numerals

## Installation

**Precondition**: node.js must be installed

```bash
npm install
```

## Testing units and integrations

```bash
npm test
```

## Testing end to end

```bash
npm run test-e2e
```

## Run

```bash
npm run start-all

# Then tape on browser : http://localhost:{port}/front/
```
